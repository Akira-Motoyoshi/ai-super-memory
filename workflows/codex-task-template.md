# Codex Task Template

Codexに作業依頼するときのテンプレート。

```md
対象リポジトリ:

目的:

やってほしいこと:
-

必ず読んでほしいファイル:
- current_context.md
- memory_index.md

必要なら読むファイル:
-

制約:
- 既存ファイルを読んでから編集する
- 破壊的に上書きしない
- 個人情報、APIキー、パスワード、メール本文の生データを保存しない
- 会話ログ全文を保存しない
- MarkdownとMermaidが壊れていないか確認する

最終出力:
- 実施内容
- 作成・更新したファイル
- 注意点
- 次にやるべきこと
- commit message案
```

## 依頼時のコツ

- 何を作るかだけでなく、何を保存しないかも書く
- 既存ファイルがある場合は「統合して」と明記する
- GitHub Pagesや就活など、関連する記憶カードを指定する
