# Memory Update Prompt

会話後に使う記憶更新プロンプト。

```md
以下の会話内容を、AI外部記憶に保存するための更新案にしてください。

条件:
- 会話ログ全文は保存しない
- 個人情報、APIキー、パスワード、メール本文の生データは保存しない
- 長期記憶、短期記憶、プロジェクト記憶に分類する
- 既存ファイルに統合する前提で、更新すべきファイルを指定する
- Mermaid mindmapの更新が必要なら提案する

出力形式:

## 1. 今日の要約
-

## 2. 保存すべき記憶
-

## 3. 更新すべきファイル
-

## 4. 追記内容
-

## 5. mindmap更新案
-

## 6. commit message
-

## 7. 保存しない方がいい情報
-
```
