# GIS / OSM

## 目的

GIS、OSM、Re:Earth、古橋研究室、YouthMappersに関する文脈を再利用しやすく整理する。

## 重要な前提

- GISやOSMを使った地図作成、データ整理、可視化に関心がある
- 研究室活動、マッピングパーティー、YouthMappersの文脈で使う
- Web地図やGitHub Pagesと組み合わせることがある

## よく使う情報

- OSMの地物、way ID、タグ、地理データ
- Re:EarthやWeb地図を使った可視化
- マッピングパーティーの活動記録
- 研究室活動や発表で使う説明
- 地図データをKML、GeoJSON、Web表示に変換する流れ

## 次回AIに読ませるべき内容

- 研究室活動を整理するときは [../projects/furuhashi-lab/activities.md](../projects/furuhashi-lab/activities.md) を読む
- 研究テーマを整理するときは [../projects/furuhashi-lab/research-themes.md](../projects/furuhashi-lab/research-themes.md) を読む
- GitHub Pagesで地図を公開するときは [github-pages.md](github-pages.md) も読む

## 更新ルール

- 活動名、目的、成果、次にやることを分けて書く
- 地図データの出典や確認状態を残す
- 個人の位置情報や非公開情報は保存しない
