# Writing Style

## 目的

回答、面接対策、文章作成で重視する表現ルールをまとめる。

## 重要な前提

- 就活では自然でわかりやすい表現を優先する
- 難しすぎる言葉や、浮いた抽象表現を避ける
- 結論ファーストで、短く話せる形にする
- 英語面接では、必要に応じて簡単な英語にする

## よく使う情報

- 面接回答は30秒程度で話せる長さを基本にする
- 「なぜその会社か」「なぜその職種か」が一文で伝わるようにする
- 抽象的な言葉は、事業、ユーザー、文化、市場、行動に落とす
- 逆質問は、事業理解、適性、成長環境を見極められるものにする

## 更新ルール

- 実際にしっくりきた言い回しを残す
- 使いにくかった表現や抽象的すぎた表現は、改善例とセットで残す
- 長文テンプレートより、短く使える言い回しを優先する
