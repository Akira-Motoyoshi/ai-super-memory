# GitHub Pages Next Actions

GitHub Pagesでサイトや地図が表示されないときの確認手順。

## チェックリスト

- [ ] GitHub Pagesの公開URLを開く
- [ ] ブラウザの開発者ツールでconsole errorを確認する
- [ ] NetworkタブでCSS、JavaScript、画像、地図ライブラリの読み込み失敗を確認する
- [ ] 地図コンテナに `height` が指定されているか確認する
- [ ] ルート相対パス `/path` ではなく、公開URLに合う相対パスになっているか確認する
- [ ] ファイル名の大文字・小文字が一致しているか確認する
- [ ] GitHub Pagesの公開ブランチと公開ディレクトリを確認する
- [ ] APIキーや外部サービスの制限が原因ではないか確認する
- [ ] 修正後にcommit/pushし、数分待って再確認する

## 解決後に残すこと

- 原因
- 修正したファイル
- 確認したURL
- 再発防止メモ
