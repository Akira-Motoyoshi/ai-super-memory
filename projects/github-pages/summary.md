# GitHub Pages Summary

## 目的

Codexで作成・修正したWebサイトをGitHub Pagesで公開し、共有できる形にする。

## 現在の課題

- Live Serverでは表示されるが、GitHub Pagesでは表示されないケースがある
- 地図表示は、CSSの高さ、相対パス、外部ライブラリ、公開設定の影響を受けやすい
- ローカルで動く状態と、GitHub上で動く状態を分けて確認する必要がある

## 管理方針

- 問題が起きたら [problems.md](problems.md) に原因候補を残す
- 次に確認することは [next-actions.md](next-actions.md) にチェックリスト化する
- 解決したら、原因、修正内容、確認方法を追記する
