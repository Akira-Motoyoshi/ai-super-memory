# Current Context

Last updated: 2026-06-22

## Active Goals

- GitHub上にAI外部記憶システムを作る
- ChatGPT/Codexの会話で無駄な前提説明を減らす
- マインドマップと記憶カードでワークフローを効率化する
- Codexでファイル更新・commit・pushできる形にする

## Active Projects

### AI Super Memory

- このリポジトリ全体を、AIに読ませる外部記憶として整える
- 会話ログ全文ではなく、要約・分類・記憶カードとして保存する
- 更新時は、既存ファイルを読んでから統合する

### GitHub Pages

- Codexで作成・修正したWebサイトをGitHub Pagesで公開できるようにする
- Live Serverでは動くがGitHub Pagesでは表示されない問題を、チェックリスト化して再利用する

### Codex活用

- リポジトリ編集、Markdown整理、GitHub Pages修正、記憶更新案作成に使う
- 作業前に既存ファイルを確認し、重複や破壊的な上書きを避ける

### 就活

- CARTA HOLDINGSなどの面接対策に使う
- マーケティング、広告、デジタル戦略、グローバル展開への関心を整理する
- 自然でわかりやすい回答を優先する

### 古橋研究室・GIS/OSM

- GIS、OSM、Re:Earth、YouthMappers、マッピングパーティーなどの活動文脈を整理する
- 研究や発表、活動報告で使える形に要約する

## ChatGPT / Codex 使用ルール

- 既に記憶にある前提を毎回繰り返さない
- 必要なファイルだけ読む
- 作業後に記憶更新案を出す
- 重要な決定は decisions/decision-log.md に残す
