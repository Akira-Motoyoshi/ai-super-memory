# Profile

## 基本情報

- AIとの会話や作業文脈を、GitHub上の外部記憶として整理して使う
- 公開されても危険が少ないように、個人情報は抽象化して記録する

## 学業・所属

- 大学で学びながら、研究室活動、GIS/OSM、Web制作、就活準備を並行して進めている
- 所属や個人を特定しすぎる情報は、必要な範囲に絞って記録する

## 研究テーマ

- GIS、OSM、Re:Earth、地理空間情報、マッピング活動に関心がある
- 研究テーマは [projects/furuhashi-lab/research-themes.md](projects/furuhashi-lab/research-themes.md) に整理する

## スキル

- Markdown整理
- GitHub / GitHub Pages運用
- Codexを使ったリポジトリ編集
- GIS / OSMを使った地図・データ整理
- 面接回答や文章の改善

## 就活の軸

- マーケティング、広告、デジタル戦略への関心
- スピード感を持って成長できる環境
- 海外でも通用する市場価値を身につけること
- 仕組みで価値を届け、人を動かす仕事への関心

## 強み

- 新しいツールを試しながら、実用に落とし込む力
- 地図、Web、AI、言語を組み合わせて考える力
- 抽象的な話を、具体的な行動や成果物に変える力

## よく使う文脈

- 就活の面接対策
- GitHub Pagesの公開・修正
- CodexでのMarkdown整理
- GIS/OSMや研究室活動の整理
- 大学課題やレポートの構成づくり

## 更新ルール

- 個人情報は詳しく書かず、再利用に必要な文脈だけ残す
- 会話ログ全文は保存しない
- 新しい文脈は、まず `current_context.md` と関連する記憶カードに反映する
- 古い情報は消す前に、必要なら `archive/` に移す
